# Round Trip Time

> [!Definiton]
> The sum of the path [[Latency|Latencies]] for a specific [[TCP protocol|TCP]] [[Segment]] to be acknowledged is called the network [[Round Trip Time]] ([[Round Trip Time|RTT]]).
