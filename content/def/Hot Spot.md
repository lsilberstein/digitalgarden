# Hot Spot

Ein Code, der häufig aufgerufen wurde. Ein Profiler prüft, ob eine Methode sehr aufgerufen wird und compiliert diese dann in nativen, optimierten Code ([[Just in Time Compiler|JiT]], [[Just in Time Compiler]]).
